One Piece Fanfic made in C++

This is an interactive text-style game coded in C++ language.
DISCLAIMER: This game is not associated with the hit anime One Piece (no copyright here).

Reminders before playing the game:
1. Your operating system should be Windows 10 or 11 (some libraries are only supported in this OS).
2. It is better to run the .exe file in Command Prompt.

How to play the game:
1. Open the .exe file
2. Press any key to move to the Main Menu
3. Press 1 to play, press 2 to quit game.
4. You are asked to enter your name and nickname.
5. The game is divided into chapters. Chapters 2 and 4 are the most important, as this allows you to choose which playstyle.
6. Two endings exist in the game. One ending continues the story to Chapter 5, which is the proper ending of the game. The other ending is when you are defeated.
7. Always remember to enjoy!
