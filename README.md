One Piece Fanfic made in C++

This is an interactive text-style game coded in C++ language.
WARNING: Work is still in progress.

Reminders when playing the game:
1. Your operating system should be Windows 10 or 11 (some libraries are only supported in this OS).
2. It is better to run the .exe file in Command Prompt.

How to play the game:
1. Open the .exe file
2. Press any key to move to the Main Menu
3. Press P to play, press Q to quit game.
4. You are asked to enter your name and nickname.
5. Play and enjoy the game!
